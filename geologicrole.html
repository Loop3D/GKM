blocksc

URI https://w3id.org/gso/geologicrole/Block
Source CGI rockbodypart vocabulary 2016-11-21
Description

Geologic unit part is one or more masses with generally sharp boundaries and block-like geometry within a matrix of some other material emplaced by processes at the earth's surface--e.g. volcanic eruption or mass wasting. Use for objects that are larger that what would be considered a clast, e.g. greater than about 10 m shortest dimension, typically large enough and sparse enough to be considered individual objects.
Super-classes gsor:Inclusionc
Sub-classes gsor:Tectonic_blockc

Clastc

URI https://w3id.org/gso/geologicrole/Clast
Description

Role for any fragment of material that is a constituent in a rock material; applies to fragments formed before lithification (e.g. sedimentary clasts) as well as fragments formed by in-place deformation of pre-existing rock (e.g. cataclastic fragments).

enclavec

URI https://w3id.org/gso/geologicrole/Enclave
Source Cobbing, 2000, Barbarin, 1991
Description

General term for a polymineralic aggregate enclosed in a granitoid. Use when origin of inclusion as xenolith or a separate igneous phase is unclear.
Super-classes gsor:Inclusionc

floating clast c

URI https://w3id.org/gso/geologicrole/Floating_Clast
Description

Clastic particles that are embedded in a sedimentary matrix in the original depositional process, e.g. clasts in a matrix-supported debris flow

Super-classes gsor:Clastc

framework clast c

URI https://w3id.org/gso/geologicrole/Framework_Clast
Description

Clasts are part of an interconnected network of clasts in a sedimentary rock. Conceptual model is the accumulation of clastic particles as originally deposited-- a heap of particles without matrix constituents.

Super-classes gsor:Clastc

framework crystalsc

URI https://w3id.org/gso/geologicrole/Framework_Crystals
Description

crystal that form the main framework of an igneous or metamorphic rock. Framework is the essential supporting structure of a rock material, formed by a collection of particles all in mutual contact to form a continuous connected framerwork; the framework commonly comprises more that one mineral phase, represented as different granular materials in this model, all in the 'framework crystal' role.
Super-classes gsor:Crystal_Rolec

geologic unit matrixc

URI https://w3id.org/gso/geologicrole/Geologic_Unit_Matrix
Source modified from CGI rockbodypart vocabulary 2016-11-21
Description

Part of a geologic unit that is generally interstitial to other constituents, e.g. in a mass wasting deposit, melange, tuff breccia.
Super-classes gsor:Matrixc

URI https://w3id.org/gso/geologicrole/Nodule

Description

An irregularly rounded mass of a mineral or mineral aggregate normally having a warty or knobby surface and no internal structure, usually with a contrasting composition from the enclosing sediment or rock matrix in which it is embedded, and that can be separated as a discrete mass from the host material.

Porphyroblast c

URI https://w3id.org/gso/geologicrole/Porphyroblast
Source https://en.wikipedia.org/wiki/Porphyroblast
Description

a large mineral crystal in a metamorphic rock which has grown within the finer grained matrix. Porphyroblasts are commonly euhedral crystals, but can also be partly to completely irregular in shape
Super-classes gsor:Holoblastc

Porphyroclast c

URI https://w3id.org/gso/geologicrole/Porphyroclast
Source Passchier C.W. & Trouw R.A.J. (2005). Microtectonics. Springer Science & Business Media. pp. 132141. ISBN 9783540293590.; https://en.wikipedia.org/wiki/Porphyroclast
Description

A clast or mineral fragment in a metamorphic rock, surrounded by a groundmass of finer grained crystals. Porphyroclasts are fragments of the original rock before dynamic recrystallisation or cataclasis produced the groundmass. This means they are older than the groundmass.
Super-classes gsor:Paleoblastc

Protolith c

URI https://w3id.org/gso/geologicrole/Protolith
Description

Role that relates a rock object to an altered, metasomatized, or metamorphic rock object produced from that rock. Non-metamorphosed rock object is role player, metamorphic product is the role object. Read X is protolith for Y.
Super-classes gsoc:Rolec
Restrictions gsoc:hasRolePlayer only gsog:Rock_Bodyc
gsoc:hasRoleObject some gsog:Rock_Bodyc
