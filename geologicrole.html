Classes

accessory constituent c

IRI https://w3id.org/gso/geologicrole/Accessory_Crystals
Description

Granular material role for constituent that forms less than about 5 percent of a Rock Material
Sub Class Of Crystal Constituent c

blocks c

IRI https://w3id.org/gso/geologicrole/Block
Description

Geologic unit part is one or more masses with generally sharp boundaries and block-like geometry within a matrix of some other material emplaced by processes at the earth's surface--e.g. volcanic eruption or mass wasting. Use for objects that are larger that what would be considered a clast, e.g. greater than about 10 m shortest dimension, typically large enough and sparse enough to be considered individual objects.
Source

CGI rockbodypart vocabulary 2016-11-21

Sub Class Of Inclusion c
Super Class Of tectonic block c

Matrix formed by deformation c

IRI https://w3id.org/gso/geologicrole/Cataclastic_Matrix
Description

rock consisting of greater than 50% sulfide or sulfosalt minerals formed by sedimentary exhalative processes.
Sub Class Of Matrix c

cement c

IRI https://w3id.org/gso/geologicrole/Cement
Description

mineral material introduced during diagenesis that binds detrital material togther in a clastic rock.

Sub Class Of Matrix c

Clast c

IRI https://w3id.org/gso/geologicrole/Clast
Description

Role for any fragment of material that is a constituent in a rock material; applies to fragments formed before lithification (e.g. sedimentary clasts) as well as fragments formed by in-place deformation of pre-existing rock (e.g. cataclastic fragments).

enclave c

IRI https://w3id.org/gso/geologicrole/Enclave
Description

General term for a polymineralic aggregate enclosed in a granitoid. Use when origin of inclusion as xenolith or a separate igneous phase is unclear.
Source

Cobbing, 2000, Barbarin, 1991
Sub Class Of Inclusion c

floating clast c

IRI https://w3id.org/gso/geologicrole/Floating_Clast
Description

Clastic particles that are embedded in a sedimentary matrix in the original depositional process, e.g. clasts in a matrix-supported debris flow

Sub Class Of Clast c

framework clast c

IRI https://w3id.org/gso/geologicrole/Framework_Clast
Description

Clasts are part of an interconnected network of clasts in a sedimentary rock. Conceptual model is the accumulation of clastic particles as originally deposited-- a heap of particles without matrix constituents.

Sub Class Of Clast c

framework crystals c

IRI https://w3id.org/gso/geologicrole/Framework_Crystals
Description

crystal that form the main framework of an igneous or metamorphic rock. Framework is the essential supporting structure of a rock material, formed by a collection of particles all in mutual contact to form a continuous connected framerwork; the framework commonly comprises more that one mineral phase, represented as different granular materials in this model, all in the 'framework crystal' role.
Sub Class Of Crystal Constituent c

geologic unit matrix c

IRI https://w3id.org/gso/geologicrole/Geologic_Unit_Matrix
Description

Part of a geologic unit that is generally interstitial to other constituents, e.g. in a mass wasting deposit, melange, tuff breccia.
Source

modified from CGI rockbodypart vocabulary 2016-11-21
Sub Class Of Matrix c

groundmass c

IRI https://w3id.org/gso/geologicrole/Groundmass
Description

in a porphyritic or porphyroblastic rock, the finer-grained cystalline material between the larger crystals

Sub Class Of Matrix c

holoblast c

IRI https://w3id.org/gso/geologicrole/Holoblast
Description

  • A crystalline grain in a metamorphic rock that is newly and completely formed during metamorphism

  • Might be a framework or accessory constituent
Source

Neuendorf et al., 2005
Sub Class Of Crystal Constituent c
Super Class Of Porphyroblast c

hydrothermal vein c

IRI https://w3id.org/gso/geologicrole/Hydrothermal_Vein
Description

A tabular or sheet-like part of a compound material formed by hydrothermal (or other metasomatic) mineral filling a fracture, may be associated with replacement of the host rock adjacent to the body
Source

Neuendorf et al 2005, CGI GTWG vocabulary
Sub Class Of Intrusive sheet c

igneous vein c

IRI https://w3id.org/gso/geologicrole/Igneous_Vein

Source

Nodule c

IRI https://w3id.org/gso/geologicrole/Nodule
Description

An irregularly rounded mass of a mineral or mineral aggregate normally having a warty or knobby surface and no internal structure, usually with a contrasting composition from the enclosing sediment or rock matrix in which it is embedded, and that can be separated as a discrete mass from the host material.

paramorph c

Source

Sub Class Of pseudomorph c

Pendant c

IRI https://w3id.org/gso/geologicrole/Pendant
Description

  • A block of wall rock material in an igneous intrusion. Pendants become xenoliths as the dimension becomes smaller than about 10 m in their longest dimension. Although term pendant has connotation of being suspended or supported from above, this is rarely demonstrable in geologic situations, and the concept here does not require connection to the wall of the containing intrusion.

  • a mass of country rock that is entirely surrounded by an igneous intrusion such as a batholith or other pluton.

Source

Base on Neuendorf et al 2005

phenocryst c

IRI https://w3id.org/gso/geologicrole/Phenocryst
Description

crystals that are larger than the enclosing groundmass in an igneous rock; typically euhedral. Phenocrysts do not play a framework role.
Sub Class Of Crystal Constituent c

Porphyroblast c

IRI https://w3id.org/gso/geologicrole/Porphyroblast
Description

a large mineral crystal in a metamorphic rock which has grown within the finer grained matrix. Porphyroblasts are commonly euhedral crystals, but can also be partly to completely irregular in shape
Source https://en.wikipedia.org/wiki/Porphyroblast
Sub Class Of holoblast c

Porphyroclast c

IRI https://w3id.org/gso/geologicrole/Porphyroclast
Description

A clast or mineral fragment in a metamorphic rock, surrounded by a groundmass of finer grained crystals. Porphyroclasts are fragments of the original rock before dynamic recrystallisation or cataclasis produced the groundmass. This means they are older than the groundmass.
Source

Passchier C.W. & Trouw R.A.J. (2005). Microtectonics. Springer Science & Business Media. pp. 132141. ISBN 9783540293590.; https://en.wikipedia.org/wiki/Porphyroclast
Sub Class Of paleoblast c

Protolith c

IRI https://w3id.org/gso/geologicrole/Protolith
Description

Role that relates a rock object to an altered, metasomatized, or metamorphic rock object produced from that rock. Non-metamorphosed rock object is role player, metamorphic product is the role object. Read X is protolith for Y.
Sub Class Of gsoc:Role

Constituent particle that has the outward form of a mineral species or particle type, but the original minerals composing the particle have been altered or replaced by different minerals, a secondary mineral whose outward crystal form has been inherited from the orignal mineral it has replaced

matrix c

IRI https://w3id.org/gso/geologicrole/Sedimentary_Matrix
Description

very fine-grained material of undiscernible sedimentary or diagenetic origin between clasts in a sedimentary rock. Includes mud matrix and pseudomatrix derived from degraded clasts e.g. in a graywacke

Sub Class Of Matrix c

syngenetic nodule c

IRI https://w3id.org/gso/geologicrole/Syngenetic_Nodule
Description

Particle formed by chemical precipitation at sediment-water interface, lacking layered structure that characterizes coated grains. Includes glauconite grains, manganese nodules, phosphate grains. Manganese nodule--\An irregular, black to brown, friable
Source

NADM SLTTs 2004, Jackson 1997
Sub Class Of Nodule c

tectonic block c

IRI https://w3id.org/gso/geologicrole/Tectonic_block
Description

Discrete masses with faulted boundaries, emplaced into the host unit by tectonic processes inside the earth, e.g. blocks in tectonic melange
Source

CGI rockbodypart vocabulary 2016-11-21

Sub Class Of blocks c

xenocryst c

IRI https://w3id.org/gso/geologicrole/Xenocryst
Description

  • A xenocryst in general can not play a framework role in a Rock Material

  • Granular material consisting of crystals that resemble phenocrysts in an igneous rock, but that are foreign to the rock in which they occur - not formed by crystallization of the magma that formed the igneous rock.
Source

Neuendorf et al., 2005
Sub Class Of Crystal Constituent c

Xenolith c

IRI https://w3id.org/gso/geologicrole/Xenolith
Description

Inclusion of pre-intrusive country rock in intrusive igneous matrix, cm to about 10 meter diameter in longest dimension. Use term pendant for larger blocks or for lithologically heterogeneous blocks.
Source

Neuendorf et al. 2005, Cobbing, 2000, This vocabulary
