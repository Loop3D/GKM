Description

Defines named ordinal eras and their older boundaries that are used to build the geologic time scale, without anchoring to particular geologic features

History Note:

2021-12-16 SMR add 'time includes' relationships so can display the hierarchy. add Pleistocene 2009 and Pliocene 2009 to account for relocation of boundary to place Gelasian in the Pleistocence. Add time contains, finishedBy and startedBy relations.
