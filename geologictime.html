Description

  • generated by S.M.Richard for Loop3D project

  • generic definition of time ordinal eras and older boundaries for eras, based on all eras defined in ISC 2020 time scale. The intention is to define generic time scale concepts to unify the materialization of those concepts in different versions of the time scale. Each specific materialization is linked to the unifying element as 'part of'.

  • Defines named ordinal eras and their older boundaries that are used to build the geologic time scale, without anchoring to particular geologic features
