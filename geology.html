Geologic Eventc # Classes

URI https://w3id.org/gso/geology/Geologic_Event
Description

When something happens, events are the ‘what’ of the happening, and processes are the ‘how’. E.g. the Trans-Hudson Orogeny (event) caused by a subduction (process). The relation between events and processes is constitution: events are constituted by processes, and processes constitute events. Geologic processes and events have at least one geologic endurant as participant, i.e. a happening cannot occur unless it happens to something. Events can only have events as parts.
Super-classes gsoc:Eventc
Restrictions gsoc:hasParticipant some (gsog:Geologic_Materialc or gsog:Geologic_Objectc)
gsoc:hasConstituent some gsog:Geologic_Processc

Geologic Time Boundaryc # Classes

URI https://w3id.org/gso/geology/Geologic_Time_Boundary
Description

A temporal feature that is the boundary between two Geologic time Intervals. Is generically dependent on a date, so can possibly change or could possibly have been a different date - e.g. the end of the Anthropocene is the current moment, which changes with each moment, and the start of the Anthropocene could have been a different time if Earth history was different. Time boundaries are hosted by a Chronostratigraphic_Contact, which has an implicit dependence on the Rock Body containing the younger Rock Body Boundary at the contact.
Super-classes gsoc:Temporal_Boundaryc
gsog:Geologic_Time_Featurec

gsoc:hostedBy some gsog:Geologic_Time_Intervalc

Geologic Time Intervalc # Classes

URI https://w3id.org/gso/geology/Geologic_Time_Interval
Description

A time interval related to the history of the Earth, having parts that are (geologic) time intervals. There are 2 primary types of geologic time intervals - these are not comprehensive (i.e. there could be additional types): generic (e.g. Jurassic), specific (e.g. Jurassic 2017). The generic time intervals define such a topology without defining concrete boundaries for the intervals. Specific intervals define boundaries anchored in the geologic record with assigned temporal positions based on some set of data (observations, e.g. age dates).
Super-classes gsog:Geologic_Time_Featurec
gsoc:Time_Interval_Featurec
Restrictions gsoc:timeFinishedBy only gsog:Geologic_Time_Featurec

Geologic time scalec # Classes

URI https://w3id.org/gso/geology/Geologic_Time_Scale
Description

A hierarchical ordinal time interval collection for which the events bounding the intervals occurred on Earth and are anchored to manifestation in the geologic record that have associated gsog:Geologic_Time_Boundaries. The part intervals must be gsog:Specific_Geologic_Time_Unit. Because the contained intervals have boundaries that are located in a temporal coodinate system, the time scale can serve as a temporal reference system.
Super-classes gsog:Geologic_Time_Interval_Collectionc
Restrictions gsoc:timeIncludes min 2 gsog:Specific_Geologic_Time_Unitc
