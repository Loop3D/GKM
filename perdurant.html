Description

Basic types of Perdurants. Perdurants (AKA occurrences) include things such as time regions, events, processes, phenomena, or activities.

Determining Event c

IRI https://w3id.org/gso/1.0/perdurant/Determining_Event
Description

An event responsible for the origing of some particular. For qualities, an observation, calculation, or inference event used to determine the value of one or more qualities. The suggested hook here is to use PROV and or SOSA vocabulary to populate Determining_Event subclasses.
