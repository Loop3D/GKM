Kinds of rock objects

Classes

Concretion c

IRI https://w3id.org/gso/1.0/rockobject/Concretion
Description

A hard, compact mass or aggregate of mineral matter, normally subspherical but commonly oblate, disc-shaped or irregular. Formed from precipitation from solution about a nucleus or centre.
Source

Neuendorf et al. 2005
Sub Class Of gsog:Specific_Rock_Object

Core c

IRI https://w3id.org/gso/1.0/rockobject/Core
Description

A planet's core, e.g. the Earth's core. Is a geologic object, because it is constituted by a mixture of geologic solid and fluid material.

Sub Class Of gsog:Geologic_Object

Crust c

IRI https://w3id.org/gso/1.0/rockobject/Crust
Description

A planet's crust, e.g. the Earth's crust. Is a rock object as it is constituted only by solid geologic material. Surprisingly, is a stratigraphic unit, because dependent spatially and temporally on the upper mantle rock object. This means it cannot be moved (e.g. to another planet) and retain identity, and it must always be younger and over the upper mantle.
Sub Class Of gsog:Geologic_Object

Crystal c

IRI https://w3id.org/gso/1.0/rockobject/Crystal
Description

An object composed of a single mineral species, with a characteristic geometrical shape, bounded by flat faces with specific, characteristic orientations reflecting a consistent, highly ordered internal arrangement of constituent atoms.
Sub Class Of gsog:Specific_Rock_Object
Restriction gsoc:hasConstituentonly

Fossil object c

IRI https://w3id.org/gso/1.0/rockobject/Fossil_Object
Description

NOTE--this class is intended to represent individual fossil objects. If some kind of material fossils are granular constituents in a sedimentary rock, they should be represented using 'Biogenic_Granular_Material, with sub classes for different kinds of fossils, e.g. in crinoid grainstone, radiolarian chert, diatomite.
Sub Class Of gsog:Specific_Rock_Object

IRI https://w3id.org/gso/1.0/rockobject/Mantle
Description
